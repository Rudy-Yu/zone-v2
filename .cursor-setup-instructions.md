### âœ… C. File Versi Runtime (Wajib!)
Buat file berikut sesuai stack:

| Stack | File | Contoh Isi |
|------|------|-----------|

| Multi-runtime | `.tool-versions` (untuk `asdf`) | `nodejs 20.12.0\npython 3.11.8` |
| Java | `.java-version` | `17` |
| .NET | `global.json` | `{ "sdk": { "version": "8.0.200" } }` |
