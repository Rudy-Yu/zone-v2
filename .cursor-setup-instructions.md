# 🚀 Standar Proyek: Sekali Install, Selamanya Jalan

Setiap proyek yang saya kerjakan harus memenuhi prinsip berikut:

> **Cukup sekali install dependensi saat pertama kali.  
> Setiap kali buka kembali — langsung bisa jalan.  
> Lingkungan harus konsisten di semua mesin developer.**

---

## 🔍 1. Deteksi Otomatis Stack Teknologi
Saat membantu setup, **deteksi dulu teknologi yang digunakan** berdasarkan file yang ada:
- `package.json` → **JavaScript/TypeScript (Node.js, React, Vue, dll)**
- `requirements.txt` / `Pipfile` / `pyproject.toml` → **Python**
- `go.mod` → **Go**
- `Cargo.toml` → **Rust**
- `pom.xml` / `build.gradle` → **Java**
- `.csproj` → **.NET**
- `pubspec.yaml` → **Flutter**
- `CMakeLists.txt` → **C/C++**

---

## 🛠️ 2. Aturan Universal untuk Semua Stack

### ✅ A. File Lock HARUS Ada & Di-Git
| Stack | File Lock yang WAJIB |
|------|------------------------|
| Node.js | `package-lock.json` |
| Yarn | `yarn.lock` |
| pnpm | `pnpm-lock.yaml` |
| Python (pip) | `requirements.txt` + (opsional) `constraints.txt` |
| Python (pipenv) | `Pipfile.lock` |
| Python (Poetry) | `poetry.lock` |
| Go | `go.sum` |
| Rust | `Cargo.lock` |
| .NET | `packages.lock.json` (aktifkan dengan `<RestorePackagesWithLockFile>true</RestorePackagesWithLockFile>`) |

> ❗ **Jangan pernah masukkan file lock ke `.gitignore`!**

### ✅ B. Folder Runtime HARUS Di-Ignore
| Folder | Alasan |
|-------|--------|
| `node_modules/` | Di-generate dari `package-lock.json` |
| `venv/`, `.venv/` | Lingkungan virtual Python |
| `__pycache__/` | Cache Python |
| `.dart_tool/`, `build/` | Flutter/React Native |
| `target/` | Rust, Java |
| `bin/`, `obj/` | .NET |
| `.next/`, `dist/`, `out/` | Build output |

### ✅ C. File Versi Runtime (Wajib!)
Buat file berikut sesuai stack:

| Stack | File | Contoh Isi |
|------|------|-----------|
| Node.js | `.nvmrc` | `20.12.0` |
| Python | `.python-version` | `3.11.8` |
| Multi-runtime | `.tool-versions` (untuk `asdf`) | `nodejs 20.12.0\npython 3.11.8` |
| Java | `.java-version` | `17` |
| .NET | `global.json` | `{ "sdk": { "version": "8.0.200" } }` |

> 💡 File ini memastikan semua developer pakai **versi runtime yang sama**.

Setiap proyek harus punya `README.md` dengan bagian:

```md
## 🚀 Jalankan Proyek

### Prasyarat
- [Deteksi otomatis: misal "Node.js v20+" atau "Python 3.11+"]

### 🔧 Pertama Kali (Hanya Sekali!)
```bash
# Pilih salah satu sesuai stack:
npm install          # Node.js
pip install -r requirements.txt  # Python (pip)
poetry install       # Python (Poetry)
dotnet restore       # .NET
flutter pub get      # Flutter
